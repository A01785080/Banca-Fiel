PASO 1: DESCARGAR
─────────────────────────────────────────────────────────────────────────
  [ ] Descarga TODOS los archivos de esta carpeta
  [ ] Crea una carpeta llamada "bancafiel-platform"
  [ ] Coloca TODOS los archivos dentro (incluidos los ocultos)

PASO 2: SUBIRLOS A GITHUB
─────────────────────────────────────────────────────────────────────────
  Abre Terminal (Mac/Linux) o CMD (Windows) y ejecuta:

    cd /ruta/a/tu/carpeta/bancafiel-platform
    git init
    git add .
    git commit -m "Inicial: BancaFiel Platform"
    git remote add origin https://github.com/TU_USUARIO/bancafiel-platform.git
    git branch -M main
    git push -u origin main

  (Reemplaza TU_USUARIO con tu usuario de GitHub)

PASO 3: ACTIVAR GITHUB PAGES
─────────────────────────────────────────────────────────────────────────
  1. Ve a GitHub → Tu repositorio
  2. Settings → Pages
  3. Branch: main
  4. Folder: / (root)
  5. Save

PASO 4: ESPERAR
─────────────────────────────────────────────────────────────────────────
  GitHub Pages tarda 2-3 minutos

PASO 5: ABRIR TU SITIO
─────────────────────────────────────────────────────────────────────────
  https://tu-usuario.github.io/bancafiel-platform
