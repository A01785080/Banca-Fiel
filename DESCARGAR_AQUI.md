## 🚀 Pasos Rápidos para Subir a GitHub

### 1️⃣ Descarga los Archivos
- Descarga todos los archivos del directorio `/outputs`
- Colócalos en una carpeta llamada `bancafiel-platform` en tu computadora

### 2️⃣ Abre Git Bash (Windows) o Terminal (Mac/Linux)
```bash
cd /ruta/a/tu/carpeta/bancafiel-platform
```

### 3️⃣ Inicializa Git
```bash
git init
git add .
git commit -m "Commit inicial: Plataforma BancaFiel"
```

### 4️⃣ Crea Repositorio en GitHub
- Ve a [github.com](https://github.com)
- Haz clic en **New Repository**
- Nombre: `bancafiel-platform`
- Privado o Público: Elige según prefieras
- **NO** inicialices con README (ya lo tienes)
- Haz clic en **Create Repository**

### 5️⃣ Conecta y Sube los Cambios
```bash
# Reemplaza TU_USUARIO con tu usuario de GitHub
git remote add origin https://github.com/TU_USUARIO/bancafiel-platform.git
git branch -M main
git push -u origin main
```

### 6️⃣ Activa GitHub Pages
- Ve a tu repositorio en GitHub
- Settings → Pages
- Branch: `main`
- Folder: `/ (root)`
- Save

**¡Tu sitio estará en vivo en 2-3 minutos!** 🎉
